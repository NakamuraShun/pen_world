# pen_world
ペンワールド片山の仮想サイト

```
git@github.com:NakamuraShun/pen_world.git
```

``
cd pen_world
```

■cake起動
```
cakephp/app/bin/cake server
```

■gulp起動
```
cd gulp
```
```
npx gulp scss_watch
```

■MySQL
起動
mysql.server start

rootログイン
mysql -uroot

■マイグレーション
```
cakephp/app/bin/cake migrations migrate
```
